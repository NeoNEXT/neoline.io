NeoLine 도움말 내용

NeoLine 지갑을 이용해 주셔서 감사합니다. 사용 중에 문제가 발생하면 다음을 찾아보십시오.

월렛 관리

  • 지갑을 만드는 방법

    【월렛】 홈페이지에서 오른쪽 상단의 스캔 버튼 옆에있는 월렛 목록 버튼을 클릭하고 월렛 생성을 선택합니다 NeoLine은 다중 지갑 디지털 자산 관리를 지원합니다.

  • 지갑을 가져 오는 방법

    현재 지갑을 가져 오는 세 가지 방법 인 개인 키, 암호화 된 키 및 NEP6을 지원합니다.


    개인 키 가져 오기 : 완전히 백업 한 개인 키를 입력하고 월렛 이름과 비밀번호를 재설정하십시오.


    NEP6 가져 오기 : Json 파일 가져 오기입니다. 개인 키를 잊어 버린 경우 Json 파일을 사용하여 복구 할 수 있지만 가져 오기를 지원하려면 먼저 월렛 비밀번호를 설정해야합니다.


    구체적인 조작 방법은 : 지갑을 열고 [지갑 가져 오기]를 클릭하고 세 번째 항목 [NEP6 파일]을 선택한 후 [파일 선택]을 클릭하면 저장된 Json 파일을 가져오고 이전에 설정 한 비밀번호를 입력하여 지갑에 로그인 할 수 있습니다 너무


    앱이 삭제되면 Android 휴대 전화에서 월렛 파일을 삭제하지 않으므로 사용자가 직접 처리해야합니다. 그러나 iPhone은 파일을 삭제하므로 다른 채널에 다른 Json 파일을 저장하는 것이 좋습니다.


    Json 파일은 다음에 구체적으로 저장됩니다.

    Android : 휴대폰 파일 관리 -NeoLine-Json 파일

    아이폰 : 휴대폰-내 아이폰-네오 라인-존슨 파일

  • 지갑을 삭제하는 방법

    지갑 목록 상자에서 해당 지갑을 선택하고 길게 눌러 삭제하십시오.

기능적 문제

  • 【월렛】, 【알아보십시오】, 【정보】 및 【광산】의 기능은 무엇입니까

    1.【월렛】 : 개인 키 백업, 지갑 내보내기, 코드 전송 스캔, 지갑 전환 및 자산 추가를 선택할 수 있습니다.

    2.【알아보십시오】 : Neo 블록 체인의 고품질 DApp가 있습니다.

    3.【정보】 : 정보는 중국어 뉴스 레터와 커뮤니티 섹션, 한국어 뉴스와 트위터 섹션으로 구분됩니다.

    4.【광산】 : 계정, 일반 설정, 이벤트 계획, 도움말 및 피드백 및 회사 정보로 구성됩니다.

  • 지갑 정보를 보는 방법

    【월렛】홈페이지에서 녹색 영역의 오른쪽에있는 3 개의 수평 점을 클릭하여 지갑 정보 페이지를 입력하면 해당 지갑의 QR 코드, 주소 및 개인 키 정보를 볼 수 있습니다.

  • 자산을 추가하는 방법

    【월렛】 홈페이지에서 자산 추가 버튼을 클릭하고 해당 자산을 추가하도록 선택하십시오.

  • 자산 거래를 수행하는 방법

    【월렛】홈페이지에서 해당 자산을 클릭하여 자산 거래 페이지로 들어가십시오. 여기서 자금을 이체하고 돈을 받고 거래 기록을 볼 수 있습니다.

보안 문제

  • 비밀번호를 잊어 버리면 어떻게합니까?

    비밀번호 검색 및 수정은 아직 지원되지 않으며 개인 키를 통해 월렛을 가져와 월렛 비밀번호를 재설정 할 수 있습니다. 그러나 비밀번호를 잊어 버리고 거래를 전송하지 않으려면【광산】로 이동하여 일반 설정에서 얼굴 또는 지문 잠금 해제를여십시오.

  • 개인 키를 잊어 버리면 어떻게합니까?

    개인 키를 잊어 버렸지 만 Json 파일을 저장하고 원래 로그인 비밀번호를 기억하는 경우 Json 파일 인 NEP6 파일을 가져 와서 개인 키를 다시 백업 할 수 있습니다.


    둘 다 분실되면 지갑의 자산을 영원히 잃게됩니다!

  • . 전송이 잘못되면 어떻게해야합니까?

    전송 기능을 사용할 때 블록 체인 작업의 "결정 불가능한"속성을 기반으로 전송 주소와 금액을 확인하십시오 오류가 발생하면 거래를 취소 할 수 없습니다.
