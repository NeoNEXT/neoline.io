NeoLine Help Center

Thank you for using NeoLine wallet, and if you have problems during using process, please check the details below:

Wallet management

  • How to create a wallet

    On the [wallet] homepage, click the list button in the top bar that next to the scan button, then choose to create a wallet. NeoLine supports multi-wallet digital asset management.

  • How to import a wallet

    Currently, NeoLine supports 3ways to import the wallet: Private Key, Encrypted Key, and NEP6 File.


    Private Key: Completely input the private key that you originally backed up, and reset the wallet name and password.


    NEP6 File: If you have forgotten your private key, you can recover it by Json file, which is the third method of the NEP6 file, but it needs the wallet password set before to assist the import.


    Specific operation method: open the wallet, click 【import wallet】-select the third item【NEP6 file】-click 【select file】, you can log in to the wallet by importing your saved Json file and entering the previously set password.


    If the app is deleted, the Android phone will not delete the wallet file, and users need to handle it by themselves. But iPhone deletes the files.


    PS. Here is where to export the wallet file:


    The Json file is stored in:

    Android: Phone-File Management-NEOLine-Json File

    iPhone: Phone-My iPhone- NEOLine-Json file

  • How to delete a wallet

    In the wallet list box, select the corresponding wallet and long hold to delete it.

Functional problems

  • What are the functions of [Wallet], [Discover], [News] and [My] respectively?

    1. [Wallet]: You can select Private Key back up, export wallet, scan code to transfer, switch wallets and add assets.

    2. [Discover]: Includes high-quality DApp in Neo Blockchain.

    3. [News]: Includes Featured and Twitter.

    4. [My]: Includes General Settings, Activity plan, Help&Feedback and About us.

  • How to view wallet information

    On the [wallet] homepage, click the three points on the lake blue main page, then enter in wallet detail page to check the corresponding wallet QR code, address, and private key information.

  • How to add assets

    On the [wallet] homepage, click the button of adding asset and select the corresponding asset.

  • How to trade assets

    On the [wallet] homepage, click on the corresponding asset and enter the asset trading page. On this page, you can transfer, receive, and view the transaction record.

  • Why has the status of transfer been in the “to be confirmed”

    1. Please wait a moment, blockchain transfer can not be received immediately. Please be patient and you usually need to wait 30 seconds .

    2. If it is to be confirmed for more than 3 minutes, it may be caused by a node failure. The record will be deleted locally, don't worry!

Safe issues

  • Forgot password, what should I do?

    Password retrieval and modification are currently not supported. You can log in to the wallet with your private key to reset your wallet password. To avoid forgetting the password cause to unable to transfer, please go to [General Settings] of [My], open Face ID unlock or Fingerprint.

  • Forgot private key, what should I do?

    Import the NEP6 file, which is the Json file, and back up the private key again. Please be sure you are saved the Json file and remember your login password.


    If you lost both (Json file and login password), you will lose your assets of wallet forever!

  • Transaction error, What should I do?

    Based on the "irrevocable" attribute of the blockchain operation, when you use the transfer function, please confirm the transfer address and the amount, If an error occurs, the transaction cannot be canceled.
