NeoLine 帮助内容

感谢您使用 NeoLine 钱包，如果您在使用过程中遇到问题，请浏览以下内容：

钱包管理

  • 如何创建钱包

    在【钱包】主页，点击右上角扫描按钮旁边的钱包列表按钮，选择创建钱包，NeoLine 支持多钱包数字资产管理。

  • 如何导入钱包

    目前支持私钥、Encrypted Key 和 NEP6 三种方式导入钱包。


    私钥导入：将您原先备份好的私钥完整输入，重新设置钱包名称和密码即可。


    NEP6 导入：也就是 Json 文件导入，如果你的私钥忘记了，可以靠 Json 文件来恢复，但需要之前设置的钱包密码进行辅助导入。


    具体操作方式为：打开钱包，点击【导入钱包】——选择第三项 【NEP6 文件】——点击【选择文件】，通过导入你保存好的 Json 文件以及输入之前设置好的密码就可以登录钱包了。


    如果删除了 App，Android 手机不会将钱包文件删除，需要用户自己进行处理。但 iPhone 会将文件删除，您最好在其他渠道额外再保存一份 Json 文件。


    Json 文件具体保存在:

    Android: 手机-文件管理- NeoLine - Json 文件

    iPhone : 手机-我的 iPhone- NeoLine - Json 文件

  • 如何删除钱包

    在钱包列表框，选择对应的钱包，长按即可删除。

功能问题

  • 【钱包】、【发现】、【资讯】、【我的】分别有什么功能

    1.【钱包】：可选择节备份私钥、导出钱包、扫码转账、切换钱包和添加资产。

    2.【发现】：Neo 区块链上优质的 DApp 都在这里。

    3.【资讯】：资讯分为两个板块，中文下的快讯、社区版块；英文下的新闻、推特版块。

    4.【我的】：由账户、通用设置、活动计划、帮助与反馈、关于我们组成。

  • 如何查看钱包信息

    在【钱包】主页，点击绿色区域右边的三个横向小点，进入钱包详情页，可查看对应钱包的二维码、地址和私钥信息。

  • 怎么添加资产

    在【钱包】主页，点击资产添加按钮，选择添加对应的资产即可。

  • 如何进行资产交易

    在【钱包】主页，点击对应资产，进入资产交易页面，可进行转账、收款，并查看交易记录

  • 转账为什么一直处于“待确认”状态

    1. 请您稍等片刻，区块链转账无法即时到账，请务必耐心等待，通常您需要等待 30 秒。

    2. 若待确认超过3分钟，则可能因为节点故障导致，该记录会在本地删除，请勿担心

安全问题

  • 忘了密码，怎们办？

    暂不支持密码找回和修改，您可以通过私钥导入钱包，重置钱包密码。但为了避免忘记密码无法转账交易，请前往【我的】，在通用设置里开启面容或者指纹解锁。

  • 忘记了私钥，怎么办？

    如果您忘记了私钥，但是保存了Json 文件以及记得原来的登录密码，您可以通过导入 NEP6 文件，也就是 Json 文件，重新备份私钥。


    若两者都丢失了，您将永远失去该钱包中的资产！

  • 转账错误怎么办？

    基于区块链操作的“不可撤销”属性，当您使用转账功能时，请您确认转账地址、金额，一旦发生错误，交易将无法取消。
